# Figure 1 caption

**Figure 1 | Structure-Mapping Agentic Memory (SMA): representation, architecture, matcher, and pre-registered evaluation.**
**(A)** A raw log session (BGL supercomputer shown) is converted by a deterministic Tier-0 encoder — rules, no learned model — into a typed predicate case: entities (slate), first-order relations (light teal), and higher-order relations (dark teal) such as `cause`, whose arguments are other statements (argument position on each arc; vertical axis = relational order). Shared sub-expressions are hash-consed, so a case is a directed acyclic graph; its canonical S-expression is content-addressed (BLAKE3), giving deduplication and a free audit trail.
**(B)** Write path: artifacts enter memory only through versioned, byte-deterministic encoders into a content-addressed case store (single source of truth) and its indexes. Read path: MAC orders candidates by an admissible content-vector bound; FAC runs full structure mapping best-first with certified early stopping; results surface as alignment receipts (correspondences, structural score, candidate inferences). A single LLM, gated outside memory (dashed), verbalizes receipts under a cite-or-abstain policy and never writes facts.
**(C)** The matcher: (1) match hypotheses seeded between same/lattice-compatible functors (dotted threads); (2) downward support closure groups them into kernels (shaded); (3) kernel conflicts (vermilion edges) define a maximum-weight-independent-set problem solved exactly by CP-SAT with logged optimality gap (selected kernels ringed); (4) trickle-down scoring (violet bars = accumulated score) makes deep, interconnected systems dominate — Gentner's systematicity as arithmetic; (5) unmatched base structure is projected into the target as a provenance-tagged candidate inference (dashed vermilion), checkable rather than generated.
**(D)** The pre-registered battery (frozen at tag `prereg-v1` before any test-seed run): tasks × methods, with confirmatory status at the time of writing. All extraction is deterministic ($0 LLM tokens).
**(E)** Confirmatory cross-system transfer (T1): label-hit@1, mean ± s.d. over five registered seeds (201–205), 200 queries per seed; index 800 sessions from the source system, query 200 from a lexically disjoint target system under a frozen ontology. Annotations give the per-query paired-bootstrap delta between SMA and the strongest baseline per leg with Holm-corrected significance (** p<0.01; n.s. not significant; 10,000 resamples). Dotted line = two-class chance. SMA wins decisively on BGL→Thunderbird, leads with parity-to-hybrids on BGL→Spirit, and — like every method — finds no signal on HDFS→OpenStack (reported as an honest null).
Abbreviations: MAC/FAC, many-are-called/few-are-chosen two-stage retrieval; CP-SAT, constraint-programming SAT solver; WAL, write-ahead log; RRF, reciprocal-rank fusion.

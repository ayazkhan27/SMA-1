- `report.html`
- `dataset_manifest.csv`
- `retrieval_runs.csv`
- `mapping_runs.csv`
- `triage_metrics.csv`
- `ssb_cases.csv`
- `ssb_metrics.csv`

- `drift_runs.csv`
- `latency.csv`
- `ablation_metrics.csv`
- `sage_pool_stats.csv`
- `calibration.csv`

The MVP report uses deterministic fixture data. Full paper runs replace pending rows with frozen
blueprint section 8 evaluation results.
